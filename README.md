# ResumeIQ - AI Resume Analyzer

ResumeIQ is an ATS Resume Analyzer built using Python and Streamlit.

## Features

* Upload Resume PDF
* ATS Score Calculation
* Skill Match Analysis
* Missing Skill Detection
* Resume Recommendations
* Skill Visualization Chart
* PDF Report Generation

## Technologies Used

* Python
* Streamlit
* Pandas
* PyPDF2
* ReportLab

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Preview

ResumeIQ helps candidates compare their resumes against job descriptions and identify missing skills to improve ATS performance.
